# Data Placeholder

The dataset used in this project is restricted and cannot be redistributed in this repository because they are controlled-access datasets and require appropriate authorization.

After obtaining the dataset with proper permission, place the processed file in this directory with the following name:

- `ADNI-longitudinal.h5`
- `OASIS3-longitudinal.h5`

Expected default path in scripts:

- `./data/ADNI-longitudinal.h5`
- `./data/OASIS3-longitudinal.h5`
